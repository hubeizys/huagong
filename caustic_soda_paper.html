一、概述

烧碱（氢氧化钠，NaOH）是造纸工业中最重要的化学试剂之一，在制浆和造纸过程中发挥着不可或缺的作用。在全球范围内，烧碱约占造纸化学品消耗量的15%，是造纸行业最重要的无机化学品。

二、制浆过程中的作用

制浆是将植物纤维原料（木材、竹子、稻草等）分离成纸浆的过程。烧碱在这一过程中起着核心化学试剂的作用。

2.1 化学反应机理

烧碱在制浆过程中的主要作用是降解木素和溶解半纤维素，从而将纤维分离出来。

// 木素降解反应（简化模型） R-CH=CH-CH₂-O-CH₃ + NaOH → R-CH=CH-CH₂-OH + NaOCH₃ // 甲氧基被羟基取代，木素大分子断裂 // 半纤维素溶解 (C₅H₁₀O₅)n + NaOH → 可溶性糖钠盐 // 半纤维素在碱性条件下水解并溶解

2.2 硫酸盐法制浆（Kraft Process）

目前最常用的制浆方法是硫酸盐法，烧碱与硫化钠配合使用：

🧪 硫酸盐法制浆工艺流程

1
原料切碎 — 将木材切成小木片，约15-25mm长
2
蒸煮液配制 — 白液含 NaOH 12-18% + Na₂S 3-5%
3
高温蒸煮 — 160-180°C，蒸煮时间4-8小时
4
纤维分离 — 木素降解，纤维被释放出来
5
黑液回收 — 燃烧黑液回收化学药品和热能
// 硫化钠的协同作用 Na₂S + H₂O → NaHS + NaOH // 生成的NaHS具有更强的脱木素能力 // 脱木素反应加速 木素 + HS⁻ → 硫代木素 + OH⁻

三、烧碱在造纸各环节的具体作用

🪵 纤维分散

帮助纤维在水中充分分散，防止纤维絮聚，保证纸页匀度

⚗️ 胶粘物控制

皂化树脂类胶粘物，使其分散或溶解于水中

🧹 清洁作用

清除设备管道中的沉积物，保持生产顺畅

📄 纸张施胶

在某些施胶工艺中作为催化剂参与反应

五、再生纸脱墨中的关键作用

在废纸回收利用中，烧碱的作用尤为重要：

  1. 油墨皂化 — 与油墨中的连接料发生皂化反应，使其从纤维上脱离
  2. 纤维分散 — 使废纸中的油墨颗粒分散成微小粒子
  3. 碱性软化 — 软化纤维素表面，提高纤维柔软度和结合力
  4. 表面清洁 — 清除纤维表面的污垢和杂质
// 脱墨过程中的皂化反应 油墨连接料(酯类) + NaOH → 肥皂(羧酸钠) + 甘油 // 酯键断裂，油墨被分散成细小颗粒

六、安全注意事项

⚠️ 烧碱安全警示

  • 强腐蚀性 — 可引起严重皮肤灼伤和眼睛损伤
  • 防护要求 — 操作时必须佩戴防护手套、护目镜和防护服
  • 禁止混合 — 切勿与酸类混合，以免产生剧烈放热反应
  • 储存要求 — 密闭保存，避免接触铝、锌等金属

七、总结

💡 关键要点

  • 烧碱是造纸工业中最重要的化学品之一
  • 核心作用：降解木素、溶解半纤维素、分离纤维
  • 主要应用：硫酸盐法蒸煮（用量最大）和废纸脱墨
  • 现代造纸采用碱回收系统，实现烧碱的循环利用，回收率可达95%以上
  • 是实现造纸工业绿色可持续发展的关键环节
